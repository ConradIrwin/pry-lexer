This is a pygments lexer for pry sessions.

Installation
============

`pip install pry_lexer`

Usage
=====

First copy-paste from your terminal into a file called `session.pry` (or anything that ends in `.pry`). Then run:

```
pygmentize -l pry session.pry
```
